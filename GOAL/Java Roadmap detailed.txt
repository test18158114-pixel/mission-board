🧩 Phase 1: Java + CS Foundations (6 Months)

🎯 Goal: Become fluent in Java syntax, OOP, and fundamentals.

📘 Topics You Must Know
	1. Java Basics

		Variables, data types (primitive + reference)
		Type casting, operators, conditionals, loops
		Functions/methods (parameters, return types, scope, static vs instance)
		Arrays (1D, 2D), Array operations
		Input/Output in Java (Scanner, BufferedReader)

	2. OOP (Object-Oriented Programming)

		Class & Object → how data + methods bind together
		Encapsulation → private data + public getters/setters
		Inheritance → extends, super, method overriding
		Polymorphism → compile-time (overloading), runtime (overriding)
		Abstraction → abstract class, interfaces
		Constructors, this, super, static keyword

	3. Exception Handling

		try, catch, finally, throw, throws
		Custom exceptions

	4. File Handling

		File creation, reading, writing (FileReader, FileWriter, BufferedReader)

	5. Multithreading

		Thread lifecycle
		Thread class vs Runnable interface
		Synchronization, inter-thread communication

	6. Collections Framework

		List (ArrayList, LinkedList)
		Set (HashSet, TreeSet)
		Map (HashMap, TreeMap)
		Queue, Stack, PriorityQueue
		Iterators, enhanced for loops
		Comparators and sorting

	7. Java 8+ Features

		Streams API, Lambda Expressions, Functional Interfaces
		Optional, Collectors, Map/Filter/Reduce

	8. Big-O Notation

		Analyze time & space complexity
		Big-O, Big-Theta, Big-Omega basics


	💡 Practice Questions (Easy → Hard)

	1️⃣ Print Fibonacci series up to N.
	2️⃣ Reverse a string without using built-in functions.
	3️⃣ Find factorial using recursion.
	4️⃣ Check if a number is prime.
	5️⃣ Count frequency of each character in a string using HashMap.
	6️⃣ Sort an array using Collections.sort().
	7️⃣ Implement a simple banking system (deposit, withdraw, balance check).
	8️⃣ Read a text file and count the number of lines/words.
	9️⃣ Create a multithreaded program to print even/odd numbers alternately.
	🔟 Use Streams API to find average salary of employees from a list.


⚙️ Phase 2: Data Structures & Algorithms (1 Year)
🎯 Goal: Be a problem-solving beast for coding rounds.

📘 Topics You Must Know
	1. Arrays & Strings

		Two pointers, sliding window
		Prefix/suffix sum, Kadane’s Algorithm
		String manipulation, substring search

	2. Linked List, Stack, Queue
		Singly & Doubly linked list
		Reversal, detection of cycle (Floyd’s algo)
		Stack using array/list
		Queue using array/list

	3. Recursion & Backtracking

		Subsets, permutations, combinations
		N-Queens, Sudoku solver

	4. Searching & Sorting

		Binary search variations (lower bound, upper bound)
		QuickSort, MergeSort, HeapSort

	5. Hashing

		HashMap, HashSet applications
		Anagrams, frequency counting

	6. Trees

		Traversals (inorder, preorder, postorder)
		Binary Search Tree operations
		Lowest Common Ancestor, Height, Diameter
		Trie construction

	7. Graphs

		BFS, DFS
		Shortest paths (Dijkstra, Bellman-Ford)
		Topological Sort, Union-Find

	8. Dynamic Programming

		Fibonacci (memoization + tabulation)
		Knapsack, LIS, LCS, Matrix DP

	💡 Practice Questions (Easy → Hard)

	1️⃣ Two Sum (LeetCode #1)
	2️⃣ Reverse a Linked List
	3️⃣ Detect Cycle in Linked List
	4️⃣ Implement Stack using Queues
	5️⃣ Search in Rotated Sorted Array
	6️⃣ Longest Substring Without Repeating Characters
	7️⃣ Lowest Common Ancestor in Binary Tree

		TCP vs UDP, HTTP/HTTPS
		Routing basics

	4. OOP + Design

		Design patterns (Singleton, Factory, Observer, Strategy)
